# Intrusion-Detector

<ul>
  <li> Django</li>
  <li> keras</li>
  <li> pandas</li>

  <li> sklearn.metrics</li>
  <li> numpy</li>
  </ul>
  <p> all these  libraries can be installed by running the following </p>
  <code> pip install -r requirements.py</code>
